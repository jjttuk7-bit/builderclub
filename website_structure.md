## 🎨 디자인 특징

### 색상 팔레트
- **배경**: 어두운 네이비-퍼플 그래디언트 (#0a0e27 → #1a1f3a)
- **주요색**: 퍼플-핑크 그래디언트 (#8b5cf6 → #ec4899)
- **텍스트**: 라이트 인디고 (#e0e6ff), 보조 텍스트 (#a8b5db)
- **액센트**: 투명한 퍼플 오버레이

### 타이포그래피
- **폰트**: Pretendard (한글 지원, 모던하고 깔끔함)
- **제목**: 900 font-weight, 3.5rem (Hero), 2rem (섹션)
- **본문**: 0.95rem-1.3rem, 600 font-weight

### 인터랙션 & 애니메이션
- 카드 호버: 상승 효과 + 발광 효과 (`translateY(-8px)`)
- 카드 진입: 스태거 애니메이션 (0.1초-0.6초 딜레이)
- 버튼 호버: 아래로 이동 + 그림자 확대
- 링크 호버: 밑줄 애니메이션

## 💡 주요 UI 컴포넌트

### 1. 카드 (Card)
- 반투명 배경 + 보더
- 상단 그래디언트 라인
- 아이콘 + 제목 + 설명 + 배지
- 호버 애니메이션

### 2. 버튼
- **Primary**: 풀 그래디언트 배경, 강한 그림자
- **Secondary**: 투명 배경, 보더만 표시

### 3. 배지 (Badge)
- 난이도 또는 카테고리 표시
- 투명 배경, 텍스트 강조색

### 4. 통계 박스 (Stat Box)
- 큰 숫자 + 라벨
- 호버 시 배경 강화

## 📊 페이지 섹션 흐름

1. **Header** - 고정 네비게이션
2. **Hero** - 임팩트 있는 인트로
3. **Builders Club** - 빌더클럽 6가지 기능 소개
4. **Claude Code** - Claude Code 5가지 학습 경로
5. **Features** - 빌더클럽 핵심 가치 4가지
6. **Stats** - 커뮤니티 통계
7. **Footer** - 저작권 정보
